观后感 vue 2018/05/28

vue个人总结（1）

1. 大概组成

  • 基础语法
  • mvvm模式
  • 组件化
  • 生命周期
  • 动画特效
  • axios：ajax数据的获取
  • vue router：多页面之间的路由
  • vuex：各个组件之间的数据共享
  • 异步组件：代码上线性能更优
  • stylus：编写前端的样式
  • 递归组件：组件自身调用自己的效果

2. 目录讲解

  • vue.js vue的库文件

3.hello world

  • 创建一个vue的实例，{}中是接受到的配置项
  • el配置项指的是实例负责管理的一个区域，el: ‘#app’，选项el属性，就是element缩写，当前Vue对象挂载到哪个标签上的语法，支持CSS选择器或者dom对象，一般用id选择器选择当前页面的标签
  • Vue的选项：data属性是自定义数据。这里只是演示了一个content属性，vue会把自定义的数据可以与html的模板数据进行绑定。
  • Vue 数据绑定的方式就是用两个花括号

4.实现一个todollist的部分功能

  • v-for指令 v-for=’item in list’ 意思是告诉vue要循环list这个数据，循环的每一项都放在item中

  • v-on指令：绑定事件；

    • 指的是在模板的button上面绑定一个事件

    • 点击button的时候要在上面执行一个函数，函数的名字叫做“handleBtnClick” 所以v-on:click=”handleBtnClick”

  • v-model=”inputValue” 指的是数据的双向绑定

5.使用组件化思想改变todolist

  • v-bind语法： 向子组件传入一个绑定值 v-bind:content=’item’
