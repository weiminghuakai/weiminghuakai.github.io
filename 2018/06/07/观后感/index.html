4.兄弟组件数据传递

  • 兄弟组件是非父子组件，非父子组件通过总线‘bus’的形式进行传值，

  • this.$emit(‘change’,e.target.innerText)，调用this.$emit()向外触发事件，事件的名字是change，事件携带的内容是e.target.innerText，

  • 向外触发事件之后，city.vue通过@change=”handleLetterChange”来监听事件，然后在methods中定义handleLetterChange事件，

  • 这样父组件就接收到子组件传递过来的数据，然后再将letter转发给子组件list.vue，父组件通过属性的形式向子组件传递数据，

  • 在data中定义一个新的数据，letter:’’，
    当接收到外部传递过来的letter的时候，this.letter = letter
    最后只需要将letter传递给citylist.vue就行了

<city-list :cities="cities" :hot="hotCities" :letter="letter"></city-list>

  • 父组件传递了letter，子组件就要接受letter，

接受letter props: { hot: Array, cities: Object, letter: String }

  • list.vue中借助watch侦听器，监听letter的变化
